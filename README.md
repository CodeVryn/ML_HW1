# Предсказание стоимости автомобилей

## Что было сделано

1. Предобработка данных:
   - Парсинг полей torque, mileage, engine, max_power, seats
   - Парсинг manufacturer из name
   - Заполнение пустых значений средними значениями

2. Модели
   - В части 1 ДЗ было проверено несколько моделей. Наилучшей оказалась ElasticNet
   - В части 2 ДЗ в файле [EDA.ipynb](notebooks/EDA.ipynb) была проведена предобработка данных (логарифмирование цены и числовых признаков) и сравнение модели без предобработки с моделью после предобработки. Предобработка улучшила качество предсказаний модели

3. Streamlit приложение
   - Было создано streamlit приложение, в которое можно загрузить csv файл и получить предсказание стоимости на основе данных из этого файла
   - Приложение было развернуто на серверах Streamlit

## Результаты

Наилучшей моделью оказалась ElasticNet с логарифмированными ценой и признаками km_driven, torque_nm, torque_rpm_min, torque_rpm_max.

Результаты на тестовой выборке:

- R2: 0.88296
- MSE: 67276570371
- MAE: 116429

Наибольший прирост качества был при:

- Переходе к ElasticNet модели
- Логарифмировании признаков и целевой переменной (цены)

## Оценка разработанного сервиса

Сервис получился удобным в использовании, с простым и понятным функционалом.
